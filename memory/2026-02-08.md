# 2026-02-08 — S4S App Build Day

## S4S Manager App

**Live at:** https://s4s-app.vercel.app

### Features Built Today
- Dashboard with real model data (15 Plush models)
- Models view with fans, earnings, LTV
- **Promo image upload** - drag & drop per model, stores in localStorage
- **Rotation schedule page** at `/rotation` - shows live ghost tag timing
- Save button on upload page
- URL-based navigation (tabs persist on refresh)

### Rotation Logic (mirrors competitor)
- 15 models = 4.3 min interval between tags
- Each model tags every other model once per hour
- Ghost tags: post → delete after 5 min
- 24hr pinned posts: `expireDays: 1` parameter
- 210 posts/hour across network
